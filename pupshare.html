PupShare

PupShare is a dog sharing site for dog lovers that can't own their own dog, but would like to spend time with one. Users can request to 'share' dogs, leave reviews about dogs they've 'shared' before, and place their own dog as available to 'share' on the site!

Technologies used:

  • HTML
  • CSS
  • Bootstrap
  • Ruby on Rails
  • Github
  • Heroku
  • PostgreSQL

I implemented user authentication, created the database models, and did error handling.
